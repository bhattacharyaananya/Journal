# The Security Sandbox: A Technical Cyber Security Journal

Welcome to my technical journal and research blog, hosted live at **[bhattacharyaananya].github.io**. 

This repository serves as both the backend source code for my static site and an open-source archive of my deep dives, lab notes, tool walk-throughs, and security research. My goal is to break down complex security concepts into highly technical, reproducible, and actionable write-ups.

---

## 🧭 Journal Index & Categories

The content on the live site is structured into four core domains:

* **`/Blue-Team-SOC`** – Incident response playbooks, SIEM logging architectures (Wazuh, Splunk), detection engineering, and threat hunting.
* **`/Offensive-Security`** – VAPT write-ups, custom exploit scripts, active defense mechanisms (honeypots), and Active Directory attack vectors.
* **`/Digital-Forensics`** – Artifact analysis, memory and disk forensics, network packet analysis (`.pcap`), and blockchain ledger tracking.
* **`/Research-Crypto`** – Explanations of cryptographic flaws, reverse engineering binaries, and emerging security paradigms.

---

## 🛠️ Tech Stack & Site Architecture

This journal is built statically and optimized for fast, distraction-free reading:

* **Framework:** Built using GitHub Pages native support for **Jekyll** *(swap to Hugo/Astro if applicable)*.
* **Styling:** Minimalist, markdown-focused theme optimized for code block readability and syntax highlighting.
* **CI/CD:** Automated deployment via **GitHub Actions** upon pushing to the `main` branch.
